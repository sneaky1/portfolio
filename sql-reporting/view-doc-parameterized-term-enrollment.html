Parameterized Term Enrollment Report

Documentation for a reusable Oracle SQL report with optional term, organization, and level-range filters.

View Query View Sample Results Back to SQL Portfolio

Overview

This sanitized Oracle SQL example demonstrates a reusable enrollment report with optional term, organization, and level-range filters.

Purpose

The report returns active participants for a selected reporting year and allows results to be narrowed by:

  • Term
  • Organization
  • Minimum level
  • Maximum level

Tables Used

  • participants
  • enrollments
  • terms
  • organizations
  • courses

SQL Concepts Demonstrated

  • Five-table INNER JOIN
  • Bind variables
  • Optional parameter filtering
  • Active-record filtering
  • Reporting-year filtering
  • Minimum and maximum range logic
  • Duplicate prevention with DISTINCT
  • Multi-column sorting

Parameters

  • :reporting_year_id
  • :term_id
  • :organization_id
  • :level_min
  • :level_max

The reporting year is required. The remaining parameters are optional.

Optional Filter Pattern

:term_id IS NULL
OR trm.term_id = :term_id

When the parameter is null, the condition does not limit the results. When a value is supplied, only matching records are returned.

Level Range Logic

  • Minimum only: returns the selected level and higher.
  • Maximum only: returns the selected level and lower.
  • Both values: returns records within the selected range.
  • Neither value: returns all levels.

Business Logic

A record is included when:

  1. The participant is active.
  2. The enrollment belongs to the selected reporting year.
  3. Every optional filter containing a value is satisfied.

Sanitization

The public version replaces:

  • Organization names and identifiers
  • Course codes and names
  • Term and reporting-year values
  • Participant information
  • System-specific table and field names

Potential Uses

  • Enrollment reports
  • Employee training rosters
  • Membership reporting
  • Course participation
  • Department assignments
  • Certification tracking
