### Chromebook Repair Management System

A web-based repair ticketing, inventory, reporting, and device-service management system built with:

- Blazor
- ASP.NET Core
- C#
- SQL Server
- Entity Framework Core
- REST APIs
- IIS
- Git and GitHub
