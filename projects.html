Featured Project

Chromebook Repair Management System

A full-stack web application for managing repair tickets, device issues, inventory, parts requests, reports, images, charges, and repair workflows.

View Full Case Study View Screenshot Gallery View GitHub Repository

Blazor • ASP.NET Core • C# • SQL Server

Project overview

I designed and developed the system to replace disconnected manual repair tracking with a centralized application. The system provides staff with tools to submit, update, search, complete, delete, print, and report on repair tickets.

Key features

  • Open and closed repair-ticket tracking
  • Parts inventory and low-stock reporting
  • Ticket image and attachment support
  • Charge, damage, and repair reporting
  • Role-based access and API authentication
  • Audit logging and secure configuration
  • Email notifications and suggestions workflow
  • Responsive Blazor user interface

Technologies

Blazor, ASP.NET Core, C#, Entity Framework Core, SQL Server, REST APIs, IIS, HTML, CSS, JavaScript, Git, and GitHub.
