Automation Case Study

PowerShell Administration Toolkit

A collection of sanitized PowerShell scripts designed to reduce repetitive administrative work, standardize technical processes, and produce useful reports.

View Toolkit README Back to Projects

Overview

Practical automation for support and administration

The toolkit demonstrates how PowerShell can be used for remote reporting, CSV analysis, bulk file operations, validation, error handling, and repeatable workflows.

Remote Disk Reporting

Measures user-profile disk usage on a remote Windows computer and exports sorted results to CSV.

Message Analysis

Reads CSV files, groups repeated messages, counts occurrences, and creates a summary report.

Bulk Image Renaming

Reads image names from a folder, creates an editable CSV mapping, previews changes, and renames files safely.

Safety and Validation

Includes input validation, error handling, duplicate protection, relative paths, and WhatIf support.

Included Scripts

Toolkit contents

Systems Administration

Get-RemoteUserDiskUsage.ps1

Uses PowerShell Remoting to scan user folders, calculate storage use, and export results.

Data Processing

Get-MessageSummary.ps1

Extracts a selected CSV column, groups duplicate messages, and ranks them by frequency.

File Automation

Rename-ImagesFromCsv.ps1

Generates a filename mapping CSV and performs validated bulk renames with optional WhatIf previews.

Identity Administration

Get-ActiveDirectoryUserAudit.ps1

Audits enabled Active Directory users and exports identity details, password status, last logon data, and group membership to CSV.

Skills Demonstrated

Technical capabilities

  • CSV import, export, grouping, and transformation

  • Administrative reporting and documentation
